Triton Robotics @ UCSD

Sentry

Sentry is a fully autonomous robot which is restricted to running on a rail. It targets all the enemy robots running close and plays as the last defense of the base. It also provides a virtual shield with 500HP to the base while it is still alive, and can only shoot 500 ammos at maximum in one round. Thus, we need to keep it alive as long as possible and also shoot accurately and wisely.

Our most recent iteration SENTRY MK. II for RMUL 2021 featuring an upgraded feed system, a variable fly-wheel system. The sentry is a fully autonomous base defense system.
